# Octoco Assessment

Two projects built as part of the Octoco technical assessment.

## Projects

### 1. Bookstore API (`bookstore-api/`)
A RESTful API for managing a bookstore inventory, built with **Express** and **TypeScript**.

See [`bookstore-api/README.md`](bookstore-api/README.md) for setup instructions.

### 2. Crypto App (`crypto-app/`)
A React + TypeScript app that displays live cryptocurrency prices using the CoinGecko API. All prices shown in ZAR.

See [`crypto-app/README.md`](crypto-app/README.md) for setup instructions.
